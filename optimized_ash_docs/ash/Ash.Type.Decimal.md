###  __ Constraints

  * `:max` \- Enforces a maximum on the value

  * `:min` \- Enforces a minimum on the value

  * `:greater_than` \- Enforces a minimum on the value (exclusive)

  * `:less_than` \- Enforces a maximum on the value (exclusive)
